# TER-571 - Settings Reorganization into 4 Super-Tabs

## Scope

- Reorganized `/settings` into exactly four super-tabs:
  - `Access Control`
  - `Master Data`
  - `Organization`
  - `Developer`
- Moved notification preferences from settings into `/account`
- Redirected `/settings/notifications` to `/account`
- Updated the avatar menu Notifications entry to route through `/account`
- Added `COGS Settings` into the Finance navigation group
- Updated user-role deeplinks to the new settings tab structure
