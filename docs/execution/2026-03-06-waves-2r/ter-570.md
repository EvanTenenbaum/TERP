# TER-570 - Inventory Gallery View (Editable) + Remove Browse Tab

## Scope

- Removed the `Browse` tab and `Browse SKU Grid` command-strip action from the Inventory workspace only.
- Added a shared `Table | Gallery` toggle to the Inventory Operations surface.
- Introduced `BatchGalleryCard` for the new gallery presentation.
- Kept gallery rendering on the existing `InventoryWorkSurface` query, filters, sorting, drawer, and quantity-adjust flow.
- Left `InventoryBrowseSlicePage` intact for Procurement usage.
