# TER-569 - Notifications Hub - Consolidate Inbox/Notifications/Alerts

## Scope

- Replaced the standalone `/notifications` page with a tabbed Notifications Hub.
- Redirected legacy `/inbox` and `/alerts` routes into the consolidated hub.
- Renamed the top-level navigation entry from `Inbox` to `Notifications`.
- Updated internal entry points so dashboards and alerts route into the new hub instead of the retired pages.

- `TER-569`
