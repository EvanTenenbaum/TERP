# Troubleshooting Guide

**Last Updated:** 2026-01-02

---

## Common Issues and Solutions

This guide covers the most common issues users encounter and how to resolve them.

---

## Login Issues

### "Invalid credentials"

**Cause:** Incorrect email or password

**Solution:**
1. Double-check your email address
2. Verify caps lock is off
3. Try resetting your password
4. Contact your administrator if issues persist

### "Account locked"

**Cause:** Too many failed login attempts

**Solution:**
1. Wait 15 minutes and try again
2. Contact your administrator to unlock

### "Session expired"

**Cause:** Inactive for too long

**Solution:**
1. Refresh the page
2. Log in again
3. Your work should be auto-saved

## Order Issues

### "Cannot create order - client not found"

**Cause:** Client doesn't exist or is archived

**Solution:**
1. Use global search to find the client
2. Check if client is archived (Settings â†’ Archived)
3. Create a new client if needed

### "Cannot confirm order - insufficient inventory"

**Cause:** Not enough stock available

**Solution:**
1. Check batch availability in Inventory
2. Reduce order quantity
3. Wait for new inventory intake
4. Contact purchasing for restock

### "Order stuck in DRAFT status"

**Cause:** Missing required fields

**Solution:**
1. Check for validation errors (red highlights)
2. Ensure client is selected
3. Ensure at least one item is added
4. Fill in all required fields

### "Cannot edit order - already shipped"

**Cause:** Order has progressed past editable state

**Solution:**
1. Create a return for corrections
2. Create a new order if needed
3. Contact manager for exceptions

## Inventory Issues

### "Batch not found"

**Cause:** Batch code incorrect or batch archived

**Solution:**
1. Verify the batch code
2. Check archived batches
3. Use search to find similar codes

### "Cannot reserve - insufficient quantity"

**Cause:** Available quantity less than requested

**Solution:**
1. Check current reservations
2. Reduce requested quantity
3. Wait for other orders to ship
4. Check other batches of same product

### "Movement requires approval"

**Cause:** Certain movements need manager sign-off

**Solution:**
1. Wait for manager approval
2. Contact your manager directly
3. Check pending approvals in your queue

### "Cannot delete batch with reservations"

**Cause:** Batch has active order reservations

**Solution:**
1. View reservations on batch detail
2. Cancel or fulfill related orders first
3. Then delete the batch

---

## Payment Issues

### "Cannot apply credit - insufficient balance"

**Cause:** Customer doesn't have enough credit

**Solution:**
1. Check customer's credit balance
2. Apply partial credit
3. Collect remaining as payment

### "Payment amount exceeds invoice"

**Cause:** Trying to pay more than owed

**Solution:**
1. Verify invoice amount
2. Reduce payment amount
3. Apply excess as credit if intended

### "Invoice not found"

**Cause:** Invoice doesn't exist or is paid

**Solution:**
1. Search by invoice number
2. Check if already paid
3. Verify client has open invoices

---

## Client Issues

### "Duplicate client warning"

**Cause:** Similar client already exists

**Solution:**
1. Review suggested matches
2. Use existing client if same
3. Proceed with new if different

### "Cannot delete client with orders"

**Cause:** Client has order history

**Solution:**
1. Archive instead of delete
2. Merge with another client if duplicate
3. Contact administrator for data cleanup

### "Credit limit exceeded"

**Cause:** Order would exceed client's credit limit

**Solution:**
1. Collect payment first
2. Request credit limit increase
3. Get manager override approval

---

## Performance Issues

### "Page loading slowly"

**Cause:** Large data set or network issues

**Solution:**
1. Refresh the page
2. Clear browser cache
3. Use filters to reduce data
4. Check your internet connection

### "Search not returning results"

**Cause:** Search index may be stale

**Solution:**
1. Try different search terms
2. Use exact matches (quotes)
3. Wait a moment and retry
4. Report if persistent

### "Form not saving"

**Cause:** Network issue or validation error

**Solution:**
1. Check for red validation errors
2. Check internet connection
3. Try saving again
4. Copy data before refreshing

---

## Browser Issues

### "Features not working"

**Cause:** Outdated browser or compatibility issue

**Solution:**
1. Update your browser to latest version
2. Try Chrome (recommended)
3. Clear cache and cookies
4. Disable browser extensions

### "Display looks wrong"

**Cause:** Zoom level or screen size

**Solution:**
1. Reset zoom to 100% (Ctrl+0)
2. Try different screen size
3. Clear browser cache

---

## Mobile Issues

### "Buttons too small"

**Cause:** Mobile view not optimized

**Solution:**
1. Use landscape orientation
2. Zoom in on specific areas
3. Use desktop mode if needed

### "Keyboard covers form"

**Cause:** Mobile keyboard behavior

**Solution:**
1. Scroll up to see field
2. Tap outside to dismiss keyboard
3. Use tab to move between fields

---

## Data Issues

### "Data seems incorrect"

**Cause:** Sync issue or data entry error

**Solution:**
1. Refresh the page
2. Check audit log for changes
3. Report to administrator

### "Missing historical data"

**Cause:** Data may be archived

**Solution:**
1. Check date range filters
2. Look in archived records
3. Contact administrator

## Error Messages

### "Something went wrong"

**Cause:** Unexpected server error

**Solution:**
1. Refresh the page
2. Try the action again
3. Note the error details
4. Report to administrator

### "Network error"

**Cause:** Connection to server lost

**Solution:**
1. Check internet connection
2. Wait and retry
3. Refresh the page

### "Permission denied"

**Cause:** You don't have access to this feature

**Solution:**
1. Contact your administrator
2. Request appropriate permissions
3. Use alternative workflow

## Reporting Issues

When reporting an issue, include:

1. **What you were doing** - Step by step
2. **What happened** - The error or unexpected behavior
3. **What you expected** - The correct behavior
4. **Screenshots** - If applicable
5. **Browser/device** - Chrome, Firefox, mobile, etc.
6. **Time** - When it occurred

Before contacting support, try:

4. **Incognito Mode** - Test without extensions
5. **Wait and Retry** - Some issues are temporary
