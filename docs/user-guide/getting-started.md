# Getting Started with TERP

**Quick Start Guide**  
**Last Updated:** 2026-01-02

---

## Welcome!

This guide will get you up and running with TERP in under 10 minutes. Follow these steps to start using the system effectively.

---

## Step 1: Login

1. Open your browser and go to [https://terp-app-b9s35.ondigitalocean.app](https://terp-app-b9s35.ondigitalocean.app)
2. Enter your email and password
3. Click **"Sign In"**

> **First time?** Contact your administrator to get your login credentials.

---

## Step 2: Explore the Dashboard

After logging in, you'll see the **Dashboard** with:

- **KPI Summary** - Key metrics at a glance (revenue, orders, inventory)
- **Quick Actions** - Common tasks like creating orders
- **Alerts** - Items needing attention (low stock, pending approvals)
- **Recent Activity** - Latest orders and updates

## Step 4: Complete Your First Tasks

### Create a Client

1. Click **Clients** in the sidebar
2. Click **"+ New Client"**
3. Enter:
   - Name (required)
   - Email or Phone
   - Address (optional)
4. Click **"Save"**

### Create an Order

1. Click **Sales ‚Üí Orders**
2. Click **"+ New Order"**
3. Select a client
4. Add products:
   - Click **"Add Item"**
   - Select product
   - Enter quantity
5. Click **"Create Order"**

### View Inventory

1. Click **Inventory** in the sidebar
2. Browse available batches
3. Click any batch to see details:
   - Quantity available
   - Cost information
   - Location
   - Photos

## Common Workflows

### Daily Sales Workflow

```
1. Check Dashboard for alerts
2. Review pending orders
3. Process new orders
4. Record payments received
5. Update inventory as needed
```

### Inventory Receiving

```
1. Go to Inventory ‚Üí Intake
2. Create new intake record
3. Enter received items
4. Complete quality check
5. Assign storage locations
```

### Customer Payment

```
1. Go to Accounting ‚Üí Payments
2. Click "Record Payment"
3. Select client and invoice
4. Enter payment details
5. Save
```
