# NAV-006: Add Workflow Queue to Main Navigation

## Implementation Guide

### Problem

The Workflow Queue page (`/workflow-queue`) exists and is fully functional, but is NOT in the main navigation. Users can only access it via:

- Direct URL
- Dashboard widget click

This makes a critical workflow management feature **invisible** to users.

### Solution

Add Workflow Queue to the Inventory navigation group.

### Files to Modify

- `client/src/config/navigation.ts`

### Implementation

```typescript
// Add after the Vendors entry (around line 125):
{
  name: "Workflow Queue",
  path: "/workflow-queue",
  icon: GitBranch, // or Kanban icon
  group: "inventory",
  ariaLabel: "Kanban board for batch workflow management",
},
```

### Also Add Icon Import

```typescript
import {
  // ... existing imports
  GitBranch, // Add this
} from "lucide-react";
```

### Acceptance Criteria

- Workflow Queue appears in Inventory navigation group
- Clicking nav item navigates to `/workflow-queue`
- Icon displays correctly
- Item respects any feature flag if needed

1 hour
