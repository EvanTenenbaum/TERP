# ST-020: Add Drizzle Schema to TypeScript Checking

## Implementation Guide

Add drizzle folder to TypeScript checking configuration.

### Steps

1. Open `tsconfig.json`
2. Add `"drizzle/**/*"` to includes array
3. Verify TypeScript checks drizzle folder
4. Test schema error detection

### Verification

Run `npm run typecheck` and verify drizzle folder is checked.
