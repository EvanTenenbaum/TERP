# Review Presentation

This folder contains the review-ready presentation for the latest spreadsheet-native proposed designs.

## Files

- `index.html`
  - full review deck for all current pages
- `REVIEW_FEEDBACK_TEMPLATE.md`
  - copyable feedback structure for Figma comments or chat

## Intended use

1. Open `index.html` locally for a slide-style review.
2. Import the deck into Figma through the local capture workflow when a commentable presentation file is needed.
3. Use `REVIEW_FEEDBACK_TEMPLATE.md` for pinned Figma comments or for chat feedback back to Codex.
