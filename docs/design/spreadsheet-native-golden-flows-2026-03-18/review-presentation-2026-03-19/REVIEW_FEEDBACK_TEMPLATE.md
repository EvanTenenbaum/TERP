# Spreadsheet-Native Review Feedback Template

Use one block per issue.

```
[artboard]
[layout|copy|interaction|visual]
[must|should|idea]
Current issue:
Desired outcome:
Suggested direction:
```

## Good examples

```
[sales-order-sheet]
[copy]
[must]
Current issue: "Whole Order Changes" is not obvious to me on first read.
Desired outcome: I should understand this is where document-wide adjustments live.
Suggested direction: Rename it or add a one-line explainer directly under the title.
```

```
[shared-primitives]
[interaction]
[idea]
Current issue: The Keys / Help trigger is useful but still easy to miss.
Desired outcome: I want keyboard help to feel obviously available.
Suggested direction: Give it a stronger accent treatment or pair it with a small "?" badge.
```

## If feedback is positive

Positive feedback is helpful too. Use the same structure and say what should stay.
