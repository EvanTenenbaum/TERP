# TERP PM Bootstraps

Generated bootstrap files for each PM-capable surface. Use the matching file whenever a client does not automatically inherit the local repo protocol or when you are handing the PM system to a hosted, remote, or newly onboarded model.

## Available Bootstraps

- `docs/agent-context/bootstrap/claude-code-cli-local.md` - Claude Code CLI (local)
- `docs/agent-context/bootstrap/codex-cli-local.md` - Codex CLI (local)
- `docs/agent-context/bootstrap/codex-mac-app.md` - Codex Mac app
- `docs/agent-context/bootstrap/hermes.md` - Hermes
- `docs/agent-context/bootstrap/claude-mac-app.md` - Claude Mac app
- `docs/agent-context/bootstrap/claude-code-cli-cloud.md` - Claude Code CLI (cloud)
- `docs/agent-context/bootstrap/claude-code-cloud.md` - Claude Code cloud
- `docs/agent-context/bootstrap/codex-cli-cloud.md` - Codex CLI (cloud)
- `docs/agent-context/bootstrap/codex-cloud.md` - Codex cloud
- `docs/agent-context/bootstrap/claude-ai.md` - claude.ai
- `docs/agent-context/bootstrap/any-llm.md` - Future new model/service

## Universal Bootstrap

- `docs/agent-context/bootstrap/any-llm.md` - default read-only bootstrap for a newly onboarded model or any surface without a validated adapter.
