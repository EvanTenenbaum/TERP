# Documents & Records

Download invoices and bills, and review transaction history in one place.

![Documents list](./images/documents.png)

## Download an Invoice or Bill

1. Open **Documents** from the navigation.
2. Use the tabs to switch between **Invoices** and **Bills**.
3. Filter by number or status if needed.
4. Select **Download PDF** next to the record you need. Files include line items and totals.

## View Transaction History

- Navigate to **Transactions** to see invoices, payments, credits, and refunds.
- Use the search box to filter by number or status.
- Totals at the top show count, total value, and last transaction date.

## FAQ

- **PDF will not open.** Check your popup blocker or download and open locally.
- **Totals seem off.** Refresh the page to pull the latest data; if it persists, notify your TERP admin.
- **Can I export a CSV?** CSV export is on the roadmap; use PDFs for now.
