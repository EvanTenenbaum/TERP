# Production Schema Validation Guide

## Overview

The schema validation tools cannot run from local development machines due to database firewall rules. This guide explains how to run validation in the production environment.

## Option 1: DigitalOcean Console (Recommended)

### Step 1: Access Production Console

1. Go to DigitalOcean App Platform
2. Navigate to TERP app
3. Open Console tab
4. You'll have a shell in the production environment
