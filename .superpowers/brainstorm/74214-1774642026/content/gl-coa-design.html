Phase 2 Key Notes

SurfaceNotable Design Choices
General Ledger
  • Account + Period + Date scoping in toolbar (not sidebar). These are the primary filters — they narrow what's in the grid.
  • Account balance shown inline in action bar (right-aligned).
  • Reference navigation in inspector — GL entries link back to their source (invoice, payment, PO). Clickable deep-links.
  • Trial Balance as collapsible support module. Shows all accounts with debit/credit totals for the selected period. Balanced indicator at bottom. Export button.
  • Post Journal Entry opens JournalEntryForm dialog (existing component).
  • Reverse Entry available for posted entries only.
Chart of Accounts
  • Group rows by account type (Asset/Liability/Equity/Revenue/Expense). AG Grid's row grouping, collapsible. Color-coded group headers.
  • Type filter tabs in action bar — clicking one collapses all other groups (or filters the grid).
  • No support modules — this is a flat reference grid. Inspector shows account detail + edit form.
  • Create/Edit via dialog — existing CreateAccountDialog pattern.
  • No KPI band or aging — not applicable to a reference list.
