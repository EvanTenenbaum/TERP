Editable Cell Rules

Editable (accent border)

  • Grade — dropdown select
  • Status — dropdown (AWAITING_INTAKE → LIVE → ON_HOLD → etc.)
  • On Hand Qty — number input → triggers Adjustment Context drawer
  • COGS — number input, direct save

Locked (muted)

  • SKU — auto-generated identifier
  • Product — catalog reference
  • Supplier — client reference
  • Reserved — calculated from orders
  • Available — calculated (on hand − reserved)
  • Age — calculated from intake date

Drawer Behavior

Trigger: Double-click on an On Hand Qty cell. The drawer slides in from the right. The cell enters edit mode simultaneously.

Reason tag required: You must select at least one reason tag before Apply is enabled. This preserves the audit compliance requirement.

Multi-edit flow: After applying, the drawer stays open if you immediately edit another row's qty — it refreshes with the new batch context. Close explicitly with × or Cancel.

Non-qty edits: Grade, Status, and COGS edits save directly on cell blur/Enter — no drawer needed. Status changes are committed with a generic "Spreadsheet status update" reason.
