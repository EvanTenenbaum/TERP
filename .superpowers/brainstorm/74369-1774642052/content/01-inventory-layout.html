Key Layout Decisions

Dashboard stats live in the toolbar as compact badges (batches, units, value, live count) — always visible, zero vertical cost.

Quantity adjustment appears as an inline panel below the grid when triggered — quick-tags for reason + free text for notes, all in one line.

Batch summary cards (4-up) appear below the grid when a row is selected — product, stock, valuation, and a "locations" link that opens the inspector to the locations section.

Locations are accessed via the inspector panel (click "3 locations →" in the summary card). Not inline — saves space for the common case where you don't need to see locations.

Gallery/Grid toggle in the toolbar — same toolbar, filters, and saved views apply to both modes. Gallery shows product images, key metrics, and quick actions per card.

Inspector panel preserved for deep batch context — full details, status change, locations sub-grid, audit events. Opens on demand.
