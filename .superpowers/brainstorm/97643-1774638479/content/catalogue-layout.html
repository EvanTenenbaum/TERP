Component Architecture

New file: SalesCatalogueSurface.tsx — replaces both SalesSheetCreatorPage and SalesSheetsPilotSurface

Zone Existing Component Change
Toolbar DraftControls, ClientCombobox Inline into toolbar row + add Autosave/Dirty status cards
Action Bar QuickViewSelector, SaveViewDialog, AdvancedFilters Compose into sheet-native action bar pattern
Inventory Browser (left) InventoryBrowser (card-based) Replace with PowersheetGrid using inventory column defs
Preview (right) SalesSheetPreview (card-based) Replace with compact PowersheetGrid + inline totals/actions
Handoff Bar (bottom) Inline button in SalesSheetPreview New: dedicated bar with Share, → SO, → Quote, Live
Status Bar WorkSurfaceStatusBar + KeyboardHintBar Reuse as-is from sheet-native pattern

Key Decisions

  • Both grids are PowersheetGrid instances — inventory browser (read-only, cell-range select) and preview (read-only, supports remove action)
  • Dirty-state gates share/convert/live — existing logic from SalesSheetPreview preserved
  • Filtering + saved views — AdvancedFilters, QuickViewSelector, SaveViewDialog composed into action bar
  • Auto-save — 30s timer from existing SalesSheetsPilotSurface, with stale-closure-safe refs
  • tRPC data layer unchanged — salesSheets.getInventory, saveDraft, getDrafts, getViews, etc.
  • Retire: SalesSheetCreatorPage, SalesSheetsPilotSurface, SheetModeToggle for sales-sheets tab
