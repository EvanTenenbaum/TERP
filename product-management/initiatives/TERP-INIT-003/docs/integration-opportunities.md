## Data Model Additions

### New Fields for `calendarEvents` Table

```typescript
// Sales-specific fields
salesSheetId?: number;      // Link to sales sheet
salesStage?: string;        // 'prospect' | 'presented' | 'negotiation' | 'closed'
expectedRevenue?: number;   // For sales pipeline visibility
reminderType?: string;      // 'sales_sheet_followup' | 'demo' | 'negotiation'

// Collections-specific fields
overdueAmount?: number;     // Amount overdue for collections events
daysPastDue?: number;       // How many days past due
collectionStage?: string;   // 'reminder' | 'followup' | 'escalation'

// Vendor-specific fields
vendorId?: number;          // Link to vendor
purchaseOrderId?: number;   // Link to PO

// Meeting confirmation fields (for client meetings)
meetingConfirmed?: boolean;     // Did the meeting happen?
meetingConfirmedBy?: number;    // User who confirmed
meetingConfirmedAt?: timestamp; // When confirmed
meetingNotes?: text;            // Notes from the meeting
meetingOutcome?: string;        // 'completed' | 'rescheduled' | 'cancelled' | 'no-show'
```
