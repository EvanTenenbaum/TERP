# MCP Server Configuration

This project uses MCP (Model Context Protocol) servers to enable AI agents to interact with external services.

## Setup Instructions

1. Copy `.mcp.json.example` to `.mcp.json`:
   ```bash
   cp .mcp.json.example .mcp.json
   ```

2. Edit `.mcp.json` and replace `your_digitalocean_api_token_here` with your actual DigitalOcean API token.

3. The `.mcp.json` file is gitignored to prevent accidentally committing secrets.

## DigitalOcean MCP Server

The DigitalOcean MCP server enables AI agents to:
- View deployment status and logs
- Monitor applications on DigitalOcean
- Trigger deployments programmatically
- Check build status and errors
- Manage database connections
- View app health metrics

## Security

**IMPORTANT:** Never commit `.mcp.json` to version control as it contains API tokens.

The actual configuration is stored in:
- Project-level: `.mcp.json` (gitignored)
- Claude Code global config: `~/.claude.json`
